# 关于有希计划（Project Yuki）

有希计划（Project Yuki）是一个番剧综合评分平台，致力于为用户提供更客观、更全面的番剧评价参考。

> “有希”二字取自《凉宫春日系列》中的角色**长门有希**（Nagato Yuki）。作为资讯统合思念体制造的联系装置，她拥有掌握和操纵海量信息的能力。有希计划也期望能如她一般，客观、精确地统合繁杂的番剧数据。

## 我们做什么

不同的番剧平台拥有各自的用户群体和打分习惯，单一平台的评分难以全面反映一部作品的真实质量。有希计划整合来自 **Bangumi**、**MyAnimeList**、**AniList** 三个主流平台的评分数据，经过归一化处理后加权计算，得出更具参考价值的综合评分。

除评分外，平台还汇总了各渠道的关注人数，以热度指标直观反映一部番剧的受众规模。
