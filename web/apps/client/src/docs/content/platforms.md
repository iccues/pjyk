# 平台数据参数

以下参数用于计算每部番剧的[综合评分与热度](/docs/metric)。

## 评分参数

| 平台        | 权重 | 评分均值 (μ) | 评分标准差 (σ) |
| ----------- | ---- | ------------ | -------------- |
| Bangumi     | 2    | 6.41         | 0.99           |
| MyAnimeList | 1    | 7.10         | 0.780          |
| AniList     | 1    | 67.5         | 9.36           |

- **权重**：该平台在综合评分中的影响力 _（考虑到本平台主要面向中文用户，因此赋予其双倍权重）_。
- **评分均值 (μ)**：该平台所有番剧的平均分，用于计算 z-score
- **评分标准差 (σ)**：该平台评分的离散程度，用于计算 z-score

## 热度参数

| 平台        | 权重 | 热度中位数 |
| ----------- | ---- | ---------- |
| Bangumi     | 2    | 807.0      |
| MyAnimeList | 1    | 16513.5    |
| AniList     | 1    | 25071.0    |

- **权重**：该平台在综合评分中的影响力 _（考虑到本平台主要面向中文用户，因此赋予其双倍权重）_。
- **热度中位数**：用于归一化的基准值，反映该平台用户规模的量级。

---

> ⚠️ 以上参数均基于特定时间点的部分样本数据统计得出，可能不完全反映各平台当前的整体分布；随着平台变化，参数存在一定的时效性限制。
